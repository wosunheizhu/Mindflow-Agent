```

⭕ 1
│  ┌──────────────────────┐

⭕ 2
│  ┌──────────────────────┐
```

## 🎬 动画效果

### 1. 脉冲动画
当前执行步骤的圆圈会有脉冲效果：
```css
animate-pulse
```

### 2. 光环效果
当前步骤有蓝色光环：
```css
ring-4 ring-blue-200
```

### 3. 颜色过渡
所有状态变化都有平滑过渡：
```css
transition-all
```

### 4. 悬停效果
卡片悬停时有阴影增强：
```css
hover:shadow-lg
```
