### 3. 全局样式更新 ✅

#### CSS 变量定义
```css
:root {
  --primary: #1e3a8a;      /* 深蓝色 */
  --primary-dark: #1e40af;
  --primary-light: #3b82f6;
  --background: #ffffff;    /* 白色 */
  --foreground: #1f2937;    /* 深灰色文字 */
  --border: #e5e7eb;        /* 浅灰边框 */
  --secondary: #f8fafc;     /* 浅灰背景 */
}
```

#### 工具类
```css
.bg-primary { background-color: var(--primary); }
.text-primary { color: var(--primary); }
.border-primary { border-color: var(--primary); }
```
