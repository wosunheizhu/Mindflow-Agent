## 3. 🎯 工具功能详解

### 🔢 数学计算器
**功能**: 执行复杂数学运算
```
示例: "计算 123 × 456 + sqrt(144)"

性能: ✅ 优秀 (< 5秒)
```

**支持运算**:
- 基本运算: +, -, ×, ÷
- 高级函数: sqrt, log, sin, cos, tan
- 常数: pi, e
- 幂运算: ^

### 🌤️ 天气查询
**功能**: 查询城市天气信息
```
示例: "北京今天天气怎么样？"
```

**返回信息**:

- 湿度
- 城市名称

```typescript
export const tools = [
  // 添加或删除工具
```

```typescript

model: "gpt-4o"           // 可选 gpt-4o-mini
```
