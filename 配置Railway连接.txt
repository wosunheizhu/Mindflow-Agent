📋 Vercel 配置步骤：

步骤1: 访问 Vercel
   https://vercel.com/dashboard

步骤2: 进入项目
   点击 "Mindflow-Agent" 项目

步骤3: 环境变量设置
   Settings → Environment Variables

步骤4: 添加环境变量
   
   点击 "Add New" 或编辑现有的 VOICE_SERVER_URL
   
   填写：
   ┌─────────────────────────────────────────────────┐
   │ Name (Key):                                     │
   │ VOICE_SERVER_URL                                │
   │                                                 │
   │ Value:                                          │
   │ https://web-production-7bbc1.up.railway.app     │
   │                                                 │
   │ 选择环境 (Environments):                        │
   │ ☑ Production                                    │
   │ ☑ Preview                                       │
   │ ☑ Development                                   │
   └─────────────────────────────────────────────────┘
   
   点击 "Save" 按钮

步骤5: 重新部署
   
   Settings页面会提示需要重新部署。
   
   点击顶部的 "Redeploy" 按钮
   或
   Deployments → 最新部署 → ⋯ → Redeploy
